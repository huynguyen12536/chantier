# Wave 2 Implementation Reports

This folder contains evidence for coding modules in Wave 2. It is separate from the immutable historical Phase 0–14 reports.

Create one folder per roadmap module using `implementation-NN/` (for example, `implementation-01/`). The module folder records each pipeline stage from `AGENTIC_EXECUTION_MANUAL.md`:

1. Planner
2. Architect
3. Developer
4. Unit
5. Integration
6. Regression
7. Review
8. ArchVal
9. BizVal
10. Documentation
11. Git Commit
12. Git Push
13. Next Module handoff

Each report records Input, Output, Evidence, Decision, Confidence, Issues, Next Step, and PASS/FAIL. A parent agent fills module reports as coding proceeds; this folder does not retroactively alter Wave 1 records.
